# Промо-парсер

## Описание
Писал промо-парсер для товарища, который работает в ретейле. Каждую неделю в ручном режиме он собирал цены конкурентов на определенные категории товаров.
Предложил ему написать парсер, чтобы сократить время сбора информации. Как итог - время сократилось с 1 дня (ручной сбор) до 20 минут (сбор с помощью скрипта).
