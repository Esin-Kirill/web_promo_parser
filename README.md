# Web-promo-parser
--------------------------------------------------
Привет! Мой первый проект. Не судите строго, 
но все комментарии и критика приветствуются.
--------------------------------------------------
ИДЕЯ.

Идея проекта (если это вообще можно назвать проектом) очень проста.
В разговоре с другом, который работает в крупной FMCG-компании, он 
рассказал мне, что ежедневно проводит мониторинг промо акций конкурентов.
Это показалось мне интересным. Уточнил несколько деталей и приступил к
выполнению.
--------------------------------------------------
ПРОЕКТ.

Проект выполнен в Jupiter Notebook на Python-3.
Использовались следующие библиотеки:
- Selenium
- BeautifulSoup
- Xlsxwriter
- Requests
- re
- datetime
- time

Результат - excel-файл с собранной информацией о промо акциях в определенных
категориях товаров у конкурентов с необходимой для "заказчика" информацией.
--------------------------------------------------
Небольшие комментарии.

Возникали некоторые сложности при реализации загрузки всех элементов
web-страницы. Я знаю, что с помощью JavaScript можно выполнить загрузку
всех элементов, но так как я не особо с ним знаком, пришлось решать проблему
с помощью "костылей" и обходных путей.
