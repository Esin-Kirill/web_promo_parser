# Промо-парсер

## Описание
Промо-парсер, написанный для товарища, который работает в ретейле. 

## Небольшая предыстория
Каждую неделю в ручном режиме друг собирал цены конкурентов на определенные категории товаров. Тупо копипастил и тратил на это целый день.
Я только начинал изучать веб-парсинг на Python и предложил для него написать парсер, чтобы сократить время сбора информации. Собрали требования, определили формат. 
Как итог: время сократилось с 1 дня (ручной сбор) до 20 минут (сбор с помощью парсера).
